# 💫 About Me:
🔭 Я веб-разработчик (Bitrix 24 / PHP / JS) из Москвы. В работе стремлюсь к идеальной архитектуре и чистому коду.
<br><br>
👯 Ищу единомышленников для крутых проектов в сфере микросервисов, Telegram-ботов и всего, что жужжит и мигает (умный дом, IoT).

🌱 Помимо основной работы постоянно экспериментирую с железом: собираю мини-кластеры из одноплатников и обвязываю их кодом.
<br><br>
🤓 На Хабре читаю научные статьи. Читаю книги по рефакторингу и архитектуре ПО («Чистый код», «Идеальный программист», «Чистая архитектура», «Идеальная работа», «Чистый Agile» — и это мои настольные книги).
